SMOOSHYROLL App

SmooshyRoll is a web application that displays the current week’s television schedule for shows airing in the United States, powered by the TVmaze API.

The app helps users quickly discover what’s airing each day and explore detailed information about their favorite shows — all in a clean, easy-to-navigate interface.

Frontend: (React)

API: TVmaze Public API

Styling: (Tailwind / CSS)

Use the following ENV::
VITE_API_BASE_URL=https://api.tvmaze.com
